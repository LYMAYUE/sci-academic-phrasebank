# SCI Academic Phrasebank for Thesis & Journal Writing

This repository contains a distilled, structured Academic Phrasebank designed for Master/PhD thesis writing and academic journal manuscript preparation. It is based on the Manchester Academic Phrasebank (Dr. John Morley, 2014) and adapted for clean, modular agentic workflows (Model Context Protocol / Agentic Skills).

## 🚀 Features

- **Thesis-Aligned Structure**: Categorized sequentially from Chapters 1 to 5 (Introduction, Literature Review, Methodology, Results, Discussion).
- **Core Academic Functions**: Includes reusable phrases for critical writing, cautious hedging, transitions, causality, examples, and introducing citations.
- **LLM-Friendly Format**: Compressed and structured with clean markdown, making it highly effective to inject into system prompts or load as an agentic skill.

## 📁 Repository Structure

- `SKILL.md`: The main skill instruction file containing 500+ structured phrases.

## 🛠️ Usage

### General Writing
Open `SKILL.md` and search for specific section tags (e.g., `1.2` for problem statements, `3.2` for method choice, `5.5` for limitations) or functional keywords (e.g., `hedging`, `criticism`, `transition`) to find appropriate academic sentence starters.

### AI Agent / MCP Integration
If you are using an AI coding assistant (like Claude Code, Antigravity, or other MCP-capable agents), mount this repository or add the folder path to your agent's config. The agent can then automatically query the phrasebank to improve the academic quality, tone, and hedging of generated sections.
